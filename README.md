### Selected Work

| Project | What it does |
| --- | --- |
| [`dify-plugin-lingua`](https://github.com/leslie2046/dify-plugin-lingua) | Language detection for Dify workflows, with ISO language-code output and optional language scoping. |
| [`dify-ragflow-plugin`](https://github.com/leslie2046/dify-ragflow-plugin) | Connects Dify to RAGFlow for datasets, documents, chunks, retrieval, and memory workflows. |
| [`dify-plugin-minio`](https://github.com/leslie2046/dify-plugin-minio) | MinIO object-storage tools for reading, uploading, listing, metadata, and bucket-backed AI workflows. |
| [`dify-plugin-juhe`](https://github.com/leslie2046/dify-plugin-juhe) | Juhe API tools for weather, exchange rates, oil and gold prices, and stock market lookups. |
| [`dify-knowledge-plugin`](https://github.com/leslie2046/dify-knowledge-plugin) | A Dify knowledge-base plugin for bringing existing knowledge operations into plugin workflows. |
